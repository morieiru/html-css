marginとpaddingとborderと幅について

基本的にはwidth + padding + borderの大きさになる

ただしborder-sizing: border-box;が指定されてる場合はwidthの幅になる。（widthの内側にborderとpaddingが入る）

下記のように全てのタグに対して当てちゃっても良い。

* {
  box-sizing: border-box;
}
